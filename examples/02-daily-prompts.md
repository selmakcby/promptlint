# Günlük Promptlar — Kötü vs İyi

Kod yazmıyorsan da aynı kural geçerli. İşte sık karşılaşılan senaryolar.

---

## 1. Mail yazma

**❌ Kötü** — Skor: 2/10
```
patrona izin maili yaz
```

**✅ İyi** — Skor: 9/10
```
İK'ya 3 günlük yıllık izin maili yaz: 15-17 Mayıs, sebep aile düğünü,
devir aldığım proje "Q2 Dashboard" — Mert toparlayacak.
Resmi ama sıcak ton, 5 cümle, selamlama dahil.
```

---

## 2. Belge özeti

**❌ Kötü** — Skor: 3/10
```
şu PDF'imi oku ve özetini ver
```

**✅ İyi** — Skor: 9/10
```
[Ekteki kira sözleşmesi].

3 şey çıkar:
1. Kiracı aleyhine en riskli 3 madde (sayfa+satır numarasıyla)
2. Standart olmayan, müzakere edilebilir 2 madde
3. "İmzalamadan önce avukatıma şunu sormalıyım..." cümlesi

Tablo formatında, 250 kelimeden fazla yazma.
```

---

## 3. Karar destekleme

**❌ Kötü** — Skor: 2/10
```
hangi laptop alayım
```

**✅ İyi** — Skor: 9/10
```
30K TL bütçeyle laptop önerisi.
Kullanım: Next.js geliştirme + 1 saat günlük YouTube çekim/edit (Final Cut yok).
Önemli: 16GB+ RAM, M-serisi çip, 1+TB SSD.
3 model öner. Her biri için tek paragraf: niye bu, eksisi ne, nereden alınır.
```

---

## 4. İçerik üretme

**❌ Kötü** — Skor: 3/10
```
LinkedIn post yaz AI hakkında
```

**✅ İyi** — Skor: 9/10
```
LinkedIn post: "Claude Code'da prompt yazmayı öğrendiğimden beri ayda
$50 yerine $5 harcıyorum" konusunda kişisel deneyim postu.

- 3 paragraf, toplam 150 kelime
- Hook: ilk cümle merak uyandırsın
- 1 spesifik örnek (gerçek rakam)
- CTA: "Sen de denedi misin?"
- Hashtag: 3 tane, alakalı, abartı yok
- Ton: konuşur gibi, "ben" perspektifi
```

---

## 5. Plan yapma

**❌ Kötü** — Skor: 2/10
```
İstanbul'a gezi planla
```

**✅ İyi** — Skor: 9/10
```
İstanbul'da 3 günlük tatil planı: Cuma akşam-Pazar gece.
2 kişi, sahil + kültür ağırlıklı, kişi başı 5K TL toplam bütçe.
Kahvaltı dahil otel listesi (3 alternatif), her gün için sabah-öğle-akşam aktivite.
Tablo formatında, ulaşım dahil.
```

---

## Formül (kod yazmayanlar için)

```
İyi prompt = HEDEF + KISITLAR + ÇIKTI FORMATI

HEDEF        : ne istediğin tek cümleyle
KISITLAR     : bütçe, tarih, ton, kapsam
ÇIKTI FORMATI: kaç madde, tablo mu paragraf mı, kelime sayısı
```
